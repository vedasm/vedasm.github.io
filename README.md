# vedasm.github.io

My personal portfolio website, built and hosted with GitHub Pages.

🔗 **Live site:** [https://vedasm.github.io](https://vedasm.github.io/)

## About

This site showcases my projects, skills, and journey as a Computer Science undergraduate and aspiring backend AI engineer. It serves as a central hub for my technical background, core software engineering philosophy, and deployed systems.

## Features

- **Dynamic Overview:** Highlighting core principles like clean code, scalable system design, and practical AI implementation.
- **Interactive Tech Arsenal:** Category-wise breakdown of languages, databases, and frameworks.
- **Project Showcase:** Deep dives into built systems with direct links to live codebases and production deployments.
- **Seamless Contact:** Direct access to professional networks and email.

## Tech Stack

- **Frontend:** HTML5, CSS3 (Minimal, high-performance styling)
- **Deployment & Hosting:** Hosted on GitHub Pages, integrated with custom domains/routing

## The Projects Documented

*   **Vector-Hire:** A scalable two-stage candidate ranking pipeline filtering 100K+ candidates using rule-based heuristics and semantic reranking with transformer embeddings. Deployed on Hugging Face Spaces.
*   **AuthGuard Password Manager:** A secure session and password management application built on Flask, bcrypt, and Flask-Login, featuring a 99.9% uptime deployment on Vercel.

## Running Locally

```bash

# Navigate into the directory

# Open index.html in your browser, or spin up a local server:
